# WikiAluno 📚

O **WikiAluno** é uma plataforma de colaboração estudantil desenvolvida para facilitar o acesso a conteúdos e gestão de estudos.
Foi pensado em estudantes que enfrentam as adversidades para iniciar seus estudos. Contamos, portanto, com um apoio psicológico e um caminho trilhado.

## 🚀 Tecnologias Utilizadas
- **Backend:** PHP 8.x
- **Banco de Dados:** MySQL (MariaDB) com PDO
- **Segurança:** Criptografia de senhas (`password_hash`), Variáveis de Ambiente (`vlucas/phpdotenv`) e proteção contra Bots (Honeypot).
- **Frontend:** HTML5 e CSS3 (Mobile First). Penso em usar React.js (eu acho)

## ⚙️ Como o projeto foi construído
O sistema conta com um fluxo de cadastro seguro, utilizando `try/catch` para tratamento de erros de banco de dados e integração com Composer para gestão de dependências de segurança. O foco principal é a integridade dos dados e a performance em sistemas baseados em Linux.
